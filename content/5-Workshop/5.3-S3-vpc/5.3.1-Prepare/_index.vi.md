# Chuẩn bị công cụ vẽ sơ đồ

### 1. Mục tiêu

Phần này chuẩn bị môi trường để vẽ lại sơ đồ **AWS CloudSOC Architecture** giống hình mẫu nhóm đã thiết kế.

### 2. Công cụ cần dùng

- **draw.io / diagrams.net**: công cụ chính để vẽ.
- **AWS Architecture Icons**: bộ icon chuẩn của AWS.
- **Trình duyệt Chrome/Edge** hoặc draw.io Desktop.
- Ảnh sơ đồ mẫu CloudSOC trong phần Workshop để đối chiếu.

### 3. Cấu hình draw.io

1. Mở [diagrams.net](https://app.diagrams.net/).
2. Chọn nơi lưu file: Device hoặc Google Drive.
3. Chọn **Create New Diagram**.
4. Chọn **Blank Diagram**.
5. Vào **More Shapes**.
6. Bật các thư viện:
   - AWS Architecture
   - Networking
   - General
   - Flowchart
7. Bấm **Apply**.

### 4. Thiết lập canvas

Khuyến nghị:

| Thuộc tính | Giá trị |
| --- | --- |
| Page size | Custom hoặc A3 Landscape |
| Grid | Bật |
| Connection style | Orthogonal |
| Font | Arial / Inter |
| Main font size | 12–14 |
| Title font size | 18–22 |

### 5. File export cuối cùng

Khi hoàn thành, nhóm nên lưu:

- `aws-cloudsoc-architecture.drawio`
- `aws-cloudsoc-architecture.png`
- `aws-cloudsoc-architecture.svg`
- `aws-cloudsoc-architecture.pdf` nếu cần chèn vào Word/PowerPoint.
