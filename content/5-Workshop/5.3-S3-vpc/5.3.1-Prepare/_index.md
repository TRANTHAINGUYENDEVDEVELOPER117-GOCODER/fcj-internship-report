# Prepare Drawing Tools

Use draw.io / diagrams.net with AWS Architecture Icons enabled. Create a blank landscape diagram, turn on grid, use orthogonal connectors, and prepare exports as `.drawio`, `.png`, `.svg`, and `.pdf`.

Recommended setup:

- AWS Architecture icon library.
- A3 landscape or custom wide canvas.
- Arial/Inter font.
- 12–14px body labels and 18–22px group titles.
