# Build the Canvas Layout

Create the main boundaries in this order:

1. AWS Cloud.
2. AWS Region: `ap-southeast-1`.
3. VPC.
4. Availability Zone.
5. Public Subnet.

Place Threat Actor and Internet on the left, the EC2 workload inside the Public Subnet, and the SOC Analyst on the right. Keep enough space for Dashboard, Logging, Threat Detection/Response, and Governance groups.
