# Draw AWS Service Groups

Create four major groups:

- **SOC Dashboard & Access:** Amplify, Cognito, API Gateway, Dashboard API Lambda, DynamoDB.
- **Logging & Evidence Storage:** CloudTrail, CloudWatch, S3, EBS Snapshot.
- **Threat Detection & Response:** GuardDuty, Security Hub, Detective, EventBridge, Step Functions, Systems Manager, Incident Response Lambda, SNS, Amazon Q Developer.
- **Security, Compliance & Governance:** IAM, IAM Policy, AWS Config, KMS.

Use consistent colors and official AWS service names.
