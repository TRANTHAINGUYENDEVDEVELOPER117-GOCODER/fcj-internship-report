# AWS Study Group Lab Sessions / Cloud Journey Workshops

### Thông tin sự kiện

| Hạng mục | Nội dung |
| --- | --- |
| Thời gian | 05/2026 – 07/2026 |
| Hình thức | Online + tự học lab |
| Vai trò | Người tham dự và thực hành |
| Nguồn học | [The First Cloud Journey](https://cloudjourney.awsstudygroup.com/) |

### Nội dung chính

- Thực hành chuỗi lab **Explore AWS Services**: AWS Account, IAM, VPC, EC2, S3, RDS, CloudWatch, CLI.
- Thực hành **Optimizing the system**: CloudFormation, VPC Peering, Transit Gateway, Lambda automation, Grafana, IAM tag-based access.
- Thực hành **Security track**: Permission Boundary, IAM Conditions, Security Hub, AWS WAF, KMS, GuardDuty, Secrets Manager.
- Thực hành **Modernize**: Lambda, API Gateway, Cognito, SQS/SNS, CloudWatch/X-Ray, Docker, ECS, CI/CD.

### Bài học rút ra

Các lab session giúp em rèn kỹ năng thực hành theo quy trình: đọc yêu cầu → triển khai → kiểm thử → ghi chú lỗi → cleanup. Với mỗi lỗi gặp phải (SG chưa mở port, IAM thiếu quyền, route table sai, port local bị chiếm), em học cách đọc log và xử lý có hệ thống hơn.

### Giá trị với báo cáo

Nội dung trong các lab session là cơ sở để em viết Worklog 12 tuần, Proposal và Workshop. Vì vậy, toàn bộ báo cáo hiện tại được trình bày theo đúng lộ trình khóa học: Foundation → Optimize → Modernize → Final Report.
